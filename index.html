Introduction

The PANDA API provides the user with root access to the system.

Take care when using the PANDA API.

Usage

Refer to the C++ or Golang references for PANDA API usage information.

C++ Reference

This is the C++ reference.

Include Panda.hpp.

#include "Panda.hpp"

You can then instantiate the class:

Panda* myPanda = new Panda{};

Initalize it:

myPanda->initialize();

Don't forget to delete it:

delete myPanda;
